# University Library Management System

A Python-based application designed to manage library operations efficiently. This project uses a modular structure with multiple Python files to handle different functionalities.

## 🚀 Features
* **Add Books:** Easily update the library inventory.
* **View Inventory:** Display a complete list of all available books.
* **Issue Books:** Track books borrowed by students.
* **Return Books:** Process returns and update the stock automatically.

## 🛠️ Tech Stack
* **Language:** Python 3.x
* **Concepts:** Modular Programming, File Imports, List Manipulation.

## 📂 Project Structure
* `main.py` - The entry point of the application.
* `add_books.py` - Logic for adding new books.
* `show_books.py` - Logic for displaying the inventory.
* `issue_book.py` - Logic for issuing books.
* `return_books.py` - Logic for returning books.
* `utils.py` - Shared data structures.
